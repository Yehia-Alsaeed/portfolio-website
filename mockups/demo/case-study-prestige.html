01The brief

A premium car showroom needs more than a brochure site: customers want to browse the fleet, reserve vehicles, and submit offers on used cars — and staff need a way to manage all of it without touching code. Prestige Motors is that platform, built end-to-end.

03Under the hood

  • Role-based access — JWT auth with role guards separating customers from showroom admins; admins approve offers and manage listings from their own dashboard.
  • Two customer flows — vehicle reservations and a used-car offer pipeline, both feeding the admin approval queue.
  • Production hardening — API rate limiting, input validation, and a Cloudinary pipeline handling vehicle image uploads.
  • TypeScript end-to-end — React + Vite + Tailwind front, Express + Mongoose back.

04Why it matters

Notebooks prove you can model; this proves you can ship. A real multi-role product with auth, payments-adjacent flows, and an admin backend — deployed, hardened, and running in the window above.
